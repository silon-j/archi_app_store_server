# django-server-template
 基于 django 最新稳定版搭建的后端服务定制化框架，解决 django 原生部分功能不好用的问题，并提供定制化开发辅助，提升开发效率

# 开发需知

本项目采用当前最新 django LTS 版本 4.2 为基础进行搭建，请确保您的开发环境满足以下要求：

- Python 3.8 或更高版本
- Django 4.2 或更高版本
